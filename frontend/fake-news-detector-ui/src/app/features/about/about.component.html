About the Project

Fake News Detector is a student-built web application that demonstrates how machine learning models can be integrated into modern web applications. The goal is to help users quickly assess whether a news claim looks more real or fake based on textual patterns.

How it works

  1. User pastes a news article or claim.
  2. The text is sent to a backend API for analysis.
  3. An ML model evaluates the content and returns a prediction.
  4. The frontend displays a label and confidence score.

Model & Data Transparency

  • Model trained on labeled news datasets from Kaggle.
  • Prediction is based on text patterns, not factual verification.
  • Confidence score represents model certainty, not absolute truth.
  • Satire, opinion pieces, or short headlines may affect accuracy.

Limitations & Disclaimer

This tool provides an automated prediction and should not be considered a replacement for professional fact-checking or journalism.

  • Results may contain false positives or negatives.
  • Not intended for medical, legal, or financial decision-making.
  • Users should verify information using trusted sources.

Frequently Asked Questions

Is this 100% accurate?

No. Machine learning models provide estimates, not guarantees.

Do you store my text?

No. Text is only used for analysis and not stored by default.

Can this detect satire?

Satirical or humorous content may sometimes be misclassified.
